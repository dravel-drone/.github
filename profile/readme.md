# 📖 프로젝트 소개

### 📄 프로젝트 배경
 
드론 촬영 규제 완화 이후 증가한 드론 여행 수요에 맞춰, 비행 가능 지역·공역·날씨·관광정보를 통합 제공하는 플랫폼입니다.
**Dravel**은 드론 사용자에게 안전한 촬영 환경과 지역 상생 여행 경험을 동시에 제공하는 종합 서비스입니다.

---

### ✨ 주요 기능

- 전국 드론스팟 정보 및 위치 기반 검색
- 공역정보(관제권/비행금지구역 등) 지도 제공 및 비행가능성 안내
- 실시간 날씨/기상 정보 제공
- 드론 여행 코스 추천
- 주변 숙박, 식당, 관광지 정보 제공 (한국관광공사 TourAPI 연동)
- 스팟/코스 리뷰, 경험 공유 및 팔로우 기반 커뮤니티 기능
